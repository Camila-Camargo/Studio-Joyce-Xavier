## Descrição do caso escolhido

Joyce Xavier é uma profissional da área da estética e beleza que oferece serviços como design de sobrancelhas, extensão de cílios, entre outras especializações. O objetivo do nosso projeto foi desenvolver um site moderno, utilizando uma estética e paleta de cores inspiradas na identidade visual já presente em sua marca profissional. O site foi criado para promover os serviços prestados, apresentar um pouco da trajetória profissional da especialista e disponibilizar informações de contato de maneira organizada e acessível.

## Necessidade identificada

Foi identificada a necessidade de fortalecer a presença digital da profissional para ampliar seu reconhecimento e centralizar informações importantes em um único local. Antes do desenvolvimento do site, as clientes dependiam principalmente das redes sociais para obter informações, o que dificultava o acesso rápido aos serviços oferecidos, formas de contato e divulgação do trabalho para novos clientes.

## Público-alvo

O público-alvo é composto principalmente por mulheres interessadas em serviços de beleza e estética, que buscam informações sobre atendimentos, especialidades da profissional, formas de contato e agendamento de serviços.

## Objetivo do site

O principal objetivo do site é apresentar os serviços oferecidos, transmitindo credibilidade e qualidade profissional, além de incentivar os visitantes a entrarem em contato para realizar agendamentos. O site também tem como finalidade ampliar a divulgação do trabalho e da profissão da especialista.

## Processo de desenvolvimento

Inicialmente, realizamos pesquisas sobre o público-alvo e analisamos referências de outros sites do segmento de beleza e estética. Em seguida, desenvolvemos wireframes e um protótipo no Figma para definir a estrutura visual e as funcionalidades do projeto.

Após essa etapa, iniciamos o desenvolvimento no Visual Studio Code utilizando HTML para a estruturação das páginas, CSS para a estilização visual e JavaScript para adicionar elementos interativos. Durante o desenvolvimento, realizamos diversos ajustes relacionados ao layout, cores, imagens, responsividade, largura, margens e experiência do usuário.

## Principais decisões do projeto

A estrutura HTML foi organizada de forma semântica, utilizando seções que facilitam a navegação e a manutenção do código, seguindo uma hierarquia lógica de informações.

Visualmente, optamos por utilizar a paleta de cores já associada à identidade visual da Joyce Xavier, buscando transmitir elegância e sofisticação sem sobrecarregar o usuário com excesso de informações.

Também foram considerados princípios básicos de acessibilidade, como contraste adequado entre elementos, organização hierárquica de títulos e utilização de textos alternativos para imagens.

Além disso, foi adicionado um carrossel de imagens para apresentar trabalhos realizados pela profissional, tornando a navegação mais dinâmica, intuitiva e agradável.

## Funcionalidade em JavaScript

Foi utilizada uma funcionalidade em JavaScript para o funcionamento dos elementos interativos presentes no site. O principal recurso implementado foi o controle de interações visuais, proporcionando uma navegação mais dinâmica para os usuários.

A utilização do JavaScript permitiu melhorar a experiência do visitante, tornando a apresentação dos conteúdos mais moderna e interativa.

## Uso de Bootstrap

O Bootstrap foi utilizado para auxiliar na construção do layout responsivo e na organização visual dos elementos da página.

A utilização desse framework permitiu otimizar o desenvolvimento do projeto, garantindo melhor adaptação do site a diferentes tamanhos de tela e dispositivos, além de contribuir para uma interface mais organizada e consistente.

## Testes realizados

Foram realizados testes dos links externos, carregamento correto das imagens, funcionamento da funcionalidade JavaScript, organização visual em diferentes navegadores, aspectos básicos de acessibilidade, publicação e acesso ao site online.

Os testes permitiram identificar e corrigir problemas relacionados ao posicionamento de elementos, responsividade e navegação.
